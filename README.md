# Customer-Behaviour-Analysis

## Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI. The project focuses on data loading, cleaning, exploratory data analysis (EDA), SQL-based analysis, dashboard creation, and report preparation for business insights.

The goal of the project is to transform raw data into meaningful insights through analytics and visualization.

---

## Dataset

* Dataset Type: CSV / Excel dataset
* Data Includes:

  * Customer/Business related information
  * Numerical and categorical attributes
  * Transactional or analytical records

The dataset was imported into Python for preprocessing and analysis.

## Project Steps

### 1. Data Loading

* Imported dataset using Pandas
* Checked dataset structure and datatypes

### 2. Exploratory Data Analysis (EDA)

* Identified missing values
* Analyzed distributions and trends
* Created visualizations for better understanding

### 3. Data Cleaning

* Removed duplicates
* Handled missing values
* Corrected inconsistent data
* Converted datatypes where required

### 4. SQL Analysis

* Imported cleaned data into MySQL
* Executed SQL queries for:

  * Filtering
  * Aggregation
  * Grouping
  * Business insights

### 5. Power BI Dashboard

* Connected processed dataset to Power BI
* Built interactive dashboards and charts
* Added KPIs and filters for analysis

### 6. Reporting & Presentation

* Created a project report summarizing insights
* Prepared presentation slides using Gamma
